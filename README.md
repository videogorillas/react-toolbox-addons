# React Toolbox Addons

### Contribution

1. Checkout project
```bash
git clone
npm install -g bumped
npm install
```
2. Create branch and make changes
3. Build
```bash
npm build
git add .
git commit -m 'new addons'
git push
```
4. Create pull request and merge into master
5. Publish

```bash
git checkout origin master
git pull
npm login
npm run patch
```
